Crater Detection Website
By Joseph Paul Cohen

This crater detection software was written with Professor Wei Ding at the University of Massachuestts. It takes as input a Portable GrayMap format (PGM) image with a known meter per pixel value. PGMs can be generated from JPEG2000 products from JPL.

Results are presented as a Portable Network Graphics (PNG) image, ArcGIS data in a zip file, and a histogram of crater sizes. The results page can be sent as a link.


There is an administration GUI (with password protection) that allows the administor to view, pause, restart, and delete detection jobs. The system uses the Quartz Enterprise Job Scheduler http://www.quartz-scheduler.org/ so jobs can be ran on multiple threads and remote systems.
